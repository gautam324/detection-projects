for beginners open the .ipynb file on jupyter notebook and train the model using the above or any other dataset u like depends on you. after implementing the whole notebook you will get a .pkl file which u can use in your flask app to make predection open app.py in pycharm or any other ide and run the app.py and for any assistance u can comment. :)
